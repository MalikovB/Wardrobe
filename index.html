The Wardrobe

This is my wardrobe, this means this is a place where I am able to experiment and learn HTML, JavaScript, and CSS. I have just began to learn how to use javascript, html, and css. To practice and improve my skills I create stuff like this, random websites and such, you could check out more of my various creations on the projects tab on the navigation bar. This website is hosted with github pages, along with all the other projects on here.

Some of the things that I plan on making in the future are basic games. I plan on starting like I did with Java and C#, by making a maze game, then progressing into a super smash bros rip off.

However to begin with that I will need to figure out how to create classes and objects and children with JS.
